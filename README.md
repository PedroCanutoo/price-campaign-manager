# Price Campaign Manager

Sistema para gerenciamento de campanhas automáticas de desconto utilizando a API oficial do Mercado Livre.

## Funcionalidades

- Autenticação OAuth
- Sincronização de anúncios
- Campanhas de desconto
- Scheduler
- Dashboard
- Logs

## Stack

- Node.js
- Express
- SQLite
- HTML
- CSS
- JavaScript
